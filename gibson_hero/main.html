This is Gibson Hero.

You can play this game using keyboard keys A, S, D, F and G. These keys correspond to the strings from top to bottom.

The objective of the game is to gather as much points as possible. Press the matching key when the note is between the two vertical lines. The closer to the left bar you get, the more points you will receive.

The game is started and stopped by pressing Enter or the Space bar. If the difficulty level gets too high, please reload the page by pressing F5.

The game will punish cheaters.
